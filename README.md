# eve-plugins-ai

Example **eve** plugin set: an `ai-desktop` **bundle** for an Ubuntu desktop —
composing a desktop, VSCode, Claude, and related dev/AI tooling.

> **Status: scaffold.** Built out after the eve v4.0 plugin contract lands. Proves
> bundle composition from a (private) external repo. See the v4.0 roadmap in the
> `eve` repo.
